# CackOps — Offer Ladder

## Core ICP
Technical founder/operator who wants a reliable bot system without spending days in config/debug loops.

## Stage 1 — Confused Starter
### Offer: Guided Setup ($149)
**Best for:** Early users who want control but need direction.

**Includes:**
- Setup call
- Live troubleshooting
- Critical config corrections
- Next-step checklist

**Pain solved:** “I can’t get this running.”

---

## Stage 2 — Functional but Fragile
### Offer: DFY Hardening ($499)
**Best for:** Users with partial setup who need reliability now.

**Includes:**
- End-to-end setup/repair
- Gateway + routing validation
- Skills/hooks/memory hardening
- Heartbeat/cron safety pass
- Final tested handoff

**Pain solved:** “It works… until it doesn’t.”

---

## Stage 3 — Team / Scale Mode
### Offer: Advanced Automation Pack (custom)
**Best for:** Teams using multiple bots/workflows.

**Includes:**
- Multi-agent orchestration patterns
- Monitoring and escalation design
- Operational runbooks
- Failure recovery flows

**Pain solved:** “I need repeatability at team scale.”

---

## Stage 4 — Operator-as-a-Service
### Offer: Managed Bot Ops ($999/mo)
**Best for:** Operators who want outcomes without babysitting infra.

**Includes:**
- Ongoing updates + compatibility checks
- Monitoring + incident response
- Optimization sprints
- Priority support

**Pain solved:** “I don’t want to maintain this myself.”
